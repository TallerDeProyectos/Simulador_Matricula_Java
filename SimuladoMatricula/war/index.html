Simulacion de Matricula USMP
El sistema simulará la matrícula de un total de 30 alumnos, para ver la cantidad de alumnos matriculados por ciclo debe seleccionar el ciclo que corresponda al curso.

Para mostrar los cursos que podría llevar un alumno se debe ingresar el código del alumno en el cuadro de busqueda.

Para mostrar los alumnos matriculados a un curso se debe ingresar el código del curso en el cuadro de busqueda.
